# 🎓 Point Web

Point Web is a classroom attendance management system developed collaboratively by me and my classmates at SENAC.

The project was created to simplify attendance tracking by connecting students and teachers through a secure attendance code system.

## 🚀 About the Project

Point Web allows teachers to register students and manage attendance efficiently during classes.

The system works through two main panels:

### 👨‍🏫 Teacher Panel

- Register students and their information
- Register the course each student is enrolled in
- Generate attendance codes for each class session
- Monitor attendance records

### 👨‍🎓 Student Panel

- Access the attendance system
- Enter the attendance code provided by the teacher
- Confirm attendance for the current class
- View personal registration information

## ⚙️ How It Works

1. The teacher registers students and their course information.
2. At the beginning of a class, the teacher generates an attendance code.
3. Students access the system and enter the generated code.
4. The system validates the code and records the student's attendance.
5. The teacher can monitor attendance records through the management panel.
